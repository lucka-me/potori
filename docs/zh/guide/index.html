# 介绍

Potori是一个基于邮件的Ingress提名管理器。它的核心功能是从Gmail邮箱中获取与Ingress提名相关的邮件，提取其中的信息，最终将各个提名的信息呈现在应用界面中。

# 与Niantic Wayfarer比较

Niantic Wayfarer平台也提供了一个候选管理器open in new window。作为官方管理器，它拥有最及时、准确的信息，更能够修改提名的名称、说明和补充信息，以及升级和撤回提名。事实上在官方管理器推出后，全面落后且为第三方的Potori确实面临着寿终正寝的命运。Potori相较官方管理器存在一些「先天性」的不足，包括：

  • 完全依赖邮件，这意味着：
    • 需要所有类型的邮件样本，分析得到准确的查询语句来获取正确的邮件
    • 无法获取邮件中不存在的信息，例如Prime提名的位置信息
    • 与实际的提交/出结果时间存在延迟
    • 邮件丢失或未发出会直接导致无法获取提名或提名信息不更新
  • 无法修改、升级和撤回提名
  • 以图片链接生成识别ID，存在极小概率的重复可能
  • 维护受开发者能力、时间和精力所限

但是经过不断的改进，现在的Potori已经拥有一些优势：

  • 提供可视化图表
  • 提供拒绝理由
  • 与玩家自制的审核插件「Brainstorming」结合，提供与审核数据关联的可视化图表
  • 更完善的渐进式网页应用体验，以及完全原生的iOS/macOS应用
  • （可能）更加友善的界面
